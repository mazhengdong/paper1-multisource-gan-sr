# GAN-based Multi-source Image Super-Resolution (Paper 1)

This repository corresponds to the first-author paper on
GAN-based multi-source image super-resolution.

## Dataset
- Dataset A (multi-source geospatial images)
- Resolution: ~11 km → 500 m

## Method
- GAN-based generator–discriminator framework
- Multi-scale feature extraction
- Spatial & channel attention

## Implementation
Core model implementation is provided in:
